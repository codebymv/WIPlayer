# STAR WORDS: An Exciting Media Player for WIP Song Ideas
Excercise in constructing a wireframe/minimuim viable product for a future full stack media player. Many music-responsive elements in a sleek modern design meant for an engaging user experience.

**Link to project:** https://wiplayer.netlify.app/

![](https://github.com/codebymv/wiplayer/blob/main/wiplayer.gif)

## How It's Made:

**Tech used:** HTML, CSS, Vanilla JavaScript

I started with a barebones concept for a media player that could provide waveform information while also being visually striking and I believe I acheived a good balance. Looking at and comparing the visualiser bands between songs helps me guage tonal balance/volume and other properties of an audio mix while having a fun UI to work in.

## Optimizations

Many improvements are coming to this project as it will be transitioning into a fullstack application. Some include:

* a working dropdown menu with access to user accounts, playlists, and more
* easy to access search bar in responsive layouts
* more songs and playlisting options
* more efficiency optimizations to reduce any user lag or hangtime being observed through the client
